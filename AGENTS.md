# Goosic repository instructions

Before working on any task in this repository, read `CODEX_HANDOFF.md`
completely. It is the source of truth for product intent, architecture, visual
rules, development commands, release procedure, integrations, and known traps.

state, architecture, UI system, integration configuration, or build behavior.
